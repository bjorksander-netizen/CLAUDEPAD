BUG YANG DITEMUKAN & DIPERBAIKI

#MasalahTingkat PerbaikanStatus
1 Modul server gagal di-import di non-Windows
import ctypes.wintypes di level modul membuat input_core.py tidak bisa dimuat sama sekali di Linux/CI, sehingga protokol mustahil diuji otomatis.		 Sedang Deteksi platform lewat hasattr(ctypes,"windll"), dengan kelas tipe pengganti untuk non-Windows. Semua fungsi injeksi input jadi no-op yang aman. selesai
2 Slider volume membanjiri server
onValueChanged mengirim paket volset pada setiap piksel pergerakan jari — bisa ratusan pesan per detik, memicu lag dan antrean WebSocket menumpuk.		 Tinggi Throttle 40 ms plus pengiriman nilai akhir saat jari dilepas, sehingga posisi slider dijamin sinkron tanpa membanjiri jaringan. selesai
3 Layar berkedip saat mengubah orientasi
ControlActivity tidak mendeklarasikan orientation pada configChanges, sehingga Activity dibuat ulang tiap kali tombol ⤢ ditekan — koneksi sempat ditinjau ulang dan tampilan berkedip.		 Sedang orientation ditambahkan ke configChanges. Rotasi kini hanya me-layout ulang tanpa membuat ulang Activity. selesai
4 Callback status koneksi bocor antar-layar
Setelah kembali dari ControlActivity, WsClient.onState masih menunjuk closure milik Activity yang sudah mati — status koneksi berhenti ter-update di layar utama dan Activity lama tertahan di memori.		 Sedang MainActivity.onResume() mendaftarkan ulang callback miliknya dan mengosongkan onMessage. selesai
